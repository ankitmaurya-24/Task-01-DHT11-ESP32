# Task-01-DHT11-ESP32

## Objective
Read temperature & humidity using DHT11 sensor on ESP32 and display on Serial Monitor.

## Components
- ESP32 DevKit
- DHT11 sensor
- Jumper wires

## Wiring
- DHT Data -> GPIO4
- VCC -> 3.3V
- GND -> GND

## How to run
1. Open the `.ino` file in Arduino IDE.
2. Select board + COM port.
3. Open Serial Monitor at 115200 baud.
